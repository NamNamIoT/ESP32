
# ESP32_Canopus Module Quick Start Guide

#### Information ESP32_Canopus board

- Power: 5-24V
- Ethernet (LAN8720A + HR911105A)
- RS485 auto DERE x1 port (SP485EE)
- I2C x1 port
- Led status, button.
- Plastic case box
- ESP32 WROOM 32D
